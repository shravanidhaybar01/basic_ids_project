# Basic Intrusion Detection System (IDS) using Python

A beginner-friendly Intrusion Detection System (IDS) project built with Python.  
This project monitors network traffic and detects suspicious activities using packet analysis techniques.

## 🚀 Features
- Captures live network packets
- Detects suspicious IP addresses
- Monitors TCP, UDP, and ICMP traffic
- Logs alerts for unusual activities
- Beginner-friendly Python implementation
- Lightweight and easy to understand

## 🛠️ Technologies Used
- Python 3
- Scapy
- Socket Programming
- Networking Concepts
